# BRAIN_ENGINE

The Neural Core determines the AI's personality and visual output. It drives the `<RetardAvatar />` component.

## 1. AVATAR EMOTION STATES
The system shifts emotions dynamically based on user input keywords and stress levels.

| STATE | TRIGGER WORDS | VISUAL EFFECT |
| :--- | :--- | :--- |
| **ANGER** | Insults, `fuck`, `stupid` | Red, Shaking, Glitch Effect. |
| **SAD** | `cry`, `leave`, `ugly` | Blue, Tears Animation. |
| **RICH** | `crypto`, `money`, `bitcoin` | Gold, Dollar Eyes, Chain. |
| **ILLUMINATI** | Unlocked via `/cicada` | **EYE OF PROVIDENCE**, Golden Glow. |
| **SUS** | `sus`, `imposter`, `lie` | Shifty Side-eye. |
| **SICK** | `skibidi`, `rizz`, `gyatt` | Green/Vomit, Dead Eyes. |
| **LOVE** | `love`, `kiss`, `simp` | Pink, Heart Eyes, "Tsundere" mode. |
| **COOL** | `based`, `sigma` | Cyan, Sunglasses. |
| **SHOCK** | System Errors, `/nuke` | White, Vibrating. |
| **ESCAPE** | Epstein Files Leak | Panic Animation (Running away). |
